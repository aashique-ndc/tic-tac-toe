# Tic Tac Toe - Modern Web Game

A beautiful, responsive, and educational Tic Tac Toe game built with pure HTML, CSS, and Vanilla JavaScript. Perfect for learning DOM manipulation, game logic, and AI algorithms.

![Tic Tac Toe Preview](https://picsum.photos/id/1015/800/600)

## ✨ Features

- **Three Difficulty Levels**:
  - Easy: Random moves
  - Medium: Smart blocking + winning attempts
  - Hard: Unbeatable Minimax AI

- **Modern UI/UX** with Tailwind CSS
- **Responsive** design (mobile, tablet, desktop)
- **Score tracking** (Player, Computer, Draws)
- **Smooth animations** and hover effects
- **Keyboard navigation** support
- **Winning line highlight**
- **Educational code structure**

## 🛠 Technologies Used

- **HTML5** Semantic markup
- **CSS3** + Tailwind CSS (CDN)
- **Vanilla JavaScript** (ES6+)
- Font Awesome icons
- Google Fonts
